## Task goal

The goal of this test task is to develop a simple Django admin panel which purpose is to manage the content for an online store and to have multi-role support.

Необходимо реализовать модели и административный интерфейс для управления:

- магазинами (shops),
- товарами (products),
- категориями (categories),

с возможностями поиска, фильтрации, сортировки и ограничениями по редактированию некоторых полей (id).
