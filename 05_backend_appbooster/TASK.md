Реплика задания AppBooster из `backend/Python/AppBooster/backend.md` репозитория `ru-test-assignments`.

---

Необходимо реализовать простейший backend‑сервис A/B‑тестирования:

- Клиентское приложение при старте отправляет запрос к API, передавая заголовок `Device-Token`.
- Сервер отвечает словарём `{experiment_key: option}`.
- Для каждого девайса выбранный вариант должен быть **детерминированным и стабильным** (всегда один и тот же).
- Реализуются как минимум два эксперимента:
  - `button_color` с тремя вариантами цветов.
  - `price` с четырьмя вариантами цен и неравномерным распределением вероятностей.

Требуется спроектировать и реализовать API и страницу статистики, показывающую распределение девайсов по вариантам в каждом эксперименте.
