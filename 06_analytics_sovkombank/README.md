## 06_analytics_sovkombank — анализ реестра и адресов (Совкомбанк Страхование)

**Направление**: analytics (Python)  
**Компания по заданию**: Совкомбанк Страхование

### Описание

Упрощённая реализация задания: скрипт на Python, который:

- приводит исходные данные «реестра» к требуемому формату (задача 1);
- по списку адресов и городов заполняет найденный город (аналог VBA‑макроса, задача 2).

Исходные файлы ожидаются в папке `data/` (нужно положить их вручную).

### Локальный запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
