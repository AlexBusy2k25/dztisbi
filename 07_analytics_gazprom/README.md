# Аналитическое задание для Газпрома

## Дата: 04.03.2026

### Задание:
Разработать дерево метрик для мониторинга эффективности работы скважин и предложить структуру дашборда.

### Решение:

В папке представлены:
- `metrics.md` - дерево метрик и их связь с КПЭ
- `dashboard_description.md` - прототип дашборда с описанием

### Ключевые выводы:

1. **Технические метрики** позволяют отслеживать эффективность работы скважин в реальном времени
2. **Процессные метрики** показывают качество работы операторов и скорость реакции
3. **Дашборд для команды** ориентирован на оперативное управление
4. **Дашборд для бизнеса** показывает достижение КПЭ и экономический эффект

### Как это влияет на КПЭ:

- Сокращение времени реакции → уменьшение недоборов нефти
- Точная диагностика → правильные мероприятия → быстрое восстановление режима
- Быстрое восстановление → снижение перерасхода электроэнергии
