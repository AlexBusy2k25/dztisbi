# 07_analytics_gazprom — анализ эффективности скважин

## Описание
Решение задания для Газпрома по анализу работы скважин.

## Функционал
- Генерация тестовых данных по скважинам
- Расчёт ключевых метрик (эффективность, недобор, перерасход)
- Визуализация результатов

## Запуск
```bash
pip install -r requirements.txt
python main.py
