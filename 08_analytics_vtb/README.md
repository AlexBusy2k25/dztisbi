## 08_analytics_vtb — анализ клиентской базы и кампании (ВТБ)

**Направление**: analytics (Python, pandas, SQLite)

### Описание

Реплика задания «Тестовое задание УЖЦК»:

- работа с SQLite‑БД `database.db` (таблицы `pl` и `campaign`);
- расчёт средних значений ЧОД, количества клиентов, оттока и эффективности кампании.

Скрипт `main.py` показывает пример подключения к базе и расчёта агрегаций.

### Локальный запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
