## 10_analytics_samokat — продуктовая и SQL‑аналитика (Самокат)

**Направление**: analytics (Power BI / SQL)

### Описание

Реплика задания `Самокат Аналитик Данных`:

- построение отчёта по данным складов/товаров/заказов;
- выделение интересных инсайтов;
- SQL‑запросы к структуре БД.

Основное решение оформляется в отчёте и SQL‑файлах; здесь хранятся постановка, скриншоты и при необходимости вспомогательные скрипты.
