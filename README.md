В этом репозитории находятся решения тестовых заданий по направлениям **backend** и **analytics**.

### Структура репозитория

- Каждый проект расположен в отдельной папке формата `"{Номер папки}_{направление}_{название компании}"`, например: `01_backend_sovkombank`.
- В каждой папке задания есть файлы:
  - `README.md` — частный README c инструкцией по запуску.
  - `TASK.md` — описание задания.
  - `SCREEN.md` — скриншоты работающего приложения/скрипта.
